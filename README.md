# Student-Marks-Prediction
# Student Marks Prediction Using Machine Learning

This mini project predicts a student's marks based on the number of study hours using Linear Regression.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Algorithm Used

Linear Regression

## Features

- Takes study hours as input
- Predicts expected marks
- Shows graph with regression line
- Beginner-friendly machine learning project

## How To Run

1. Open the notebook in Google Colab.
2. Run all cells.
3. Enter study hours when asked.
4. The model will predict expected marks.

## Project Output

The model predicts student marks based on study hours using a trained Linear Regression model.
